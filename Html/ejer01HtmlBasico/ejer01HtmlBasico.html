Cuerpo principal del contenido

La etiqueta main es en HTML5 una etiqueta semantica. Esto quiere decir que tiene que ver con el significado de lo contendio dentro de ella. Define el contenido principal de la página. Unicamente puede haber una sola etiqueta como esta en todo el documento.
El contendio escrito aquí dentro no se repite en el resto del sitio (todo lo contrario a los encabezados de página, los menues de navegación y los pies de página). Es usado por buscadores para indexar.
Conviene usar aqui textos que ayuden a encontrar la página luego de procesos de indexación de los buscadores.

Sección etiquetas semánticas estructurales más importantes

  • HEADER
  • FOOTER
  • SECTION
  • ARTICLE
  • NAV
  • ADDRESS
Las etiquetas semánticas estructurales estan relacionadas directamente con el significado del contenido y no con los estilos ni las posiciones dentro del documento.

Reglas para etiquetas semánticas

  • Los distintos tipos de contenido se pueden asociar a las etiquetas SECTION
  • Las unidades de contenido se asocian a las etiquetas ARTICLE. O sea que un SECTION anida multiples ARTICLES
  • Las etiquetas HEADER Y FOOTER pueden ser anidadas dentro de otras HEADER o FOOTER. Osea un FOOTER podría anidar otro HEADER y FOOTER.
  • La etiqueta NAV debe ser unica en el documento y corresponde a los elementos de navegación del documento.
  • Reservar etiqueta h1 para usar dentro de las etiquetas top level como MAIN, HEADER o FOOTER
