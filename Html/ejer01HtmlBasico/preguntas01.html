  1. Dar un par de ejemplos de uso de etiquetas meta.
  2. Nombrar al menos 2 atributos aplicables a la etiqueta HTML
  3. Como se relaciona un header http con la etiqueta head de HTML?
  4. Que es UTF8?
  5. Que atributo HTML sirve para aplicar estilo, a que etiqueta es aplicable?
  6. Donde se indica que un enlace debe ubicar el documento en una nueva pestaña?
  7. Explicar cuando usarían etiquetas semánticas y con que propósito.
  8. ¿Que atributos de estilo básicos aplicarían a una tabla? Explicar.
  9. ¿Que diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP?
  10. ¿Que atributos marcan la agrupación de celdas de una tabla?. ¿A que etiquetas se aplican?

Respuestas

  1. En la cabecera del documento se pueden incluir scripts de diferentes lenguajes de programación.
    Ejemplos:
    JavaScript --> <'script> src="http://ajax.googleapis.com/ajax/libs/jquery/1.3.2/jquery.min.js" type="text/javascript"> <'script>
    CSS --> <'link rel="stylesheet" href="ruta/estilo.css" type="text/css" media="screen" />
  2. alt = Texto alternativo que se mostrará en el navegador en caso de que la imagen no se pueda mostrar.
    class = Otorga una clase creada en CSS a un elemento HTML. Se usa principalmente para aplicar estilos a elementos con propiedades en común.
  3. El elemento <'head> dota de metadatos al documento, mientras que la etiqueta <'header> representa elementos de navegación o introductorios.
  4. UTF-8 es un formato estándar para almacenar caracteres Unicode. Le indica al navegador que su contenido o valor es el código de lenguaje español utf-8
  5. Es el atributo class y es un atributo global, se puede aplicar a todos los elementos HTML
  6. target="_blank"
  7. La etiqueta HEADER, que debe englobar aquellos elementos correspondientes a la cabecera de la página.
    FOOTER, que debe englobar todas las etiquetas que componen el pie.

    8. La etiqueta NAV para espeficiar que es la barra de navegación
  8. Los atributos height (alto) y width (ancho) para el tamaño de las celdas. border="1" que se tiene que aplicar en la tabla, background-color para dar color al fondo. border-collapse para fusionar los bordes de la tabla
  9. El elemento <'body> de HTML representa el contenido de un documento HTML.
    El cuerpo de la petición HTTP es donde colocamos información adicional que vamos a enviar al servidor
  10. Para agrupar filas en una tabla html vamos a utilizar el atributo rowspan de las celdas. El atributo rowspan indica el número de filas a las cual se expande su contenido. Se puede aplicar a las etiquetas <'td>, <'th>
