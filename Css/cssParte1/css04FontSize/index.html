Tamaño de los fonts

Los textos marcados con "H1" tendrán dimención doble a los párrafos comunes. Correspónderían títulos

El ejercicio se definirá un tañamo de letra default para el body y luego todo será expresado en unidades em (unidad de referencia)

Este ejercicio se trabajará con control de medio @media() para que el tamaño de letra sea modificado según el tamaño del viewport

Dispositivos moviles de alta resolución tienen una definición mayor a 300ppi (Pixels/Pulgada)

Dispositivos moviles de baja resolución tienen una definición menor en el orden de los 100ppi (Pixels/Pulgada)

Dispositivos de pantalla grande tienen una definición en el orden de los 60 (Pixels/Pulgada)

En un movil de alta resolución estaría viendo las letras de 3 a 5 veces más chicas que en una pantalla grande si su tamaño es fijo en pixeles

Por este motivo es que si no uso el control de medio, será más conveniente definir el tamaño de letra considerando el peor de los casos (Dispositivos de alta resolución)

En este ejercicio hago crecer la letra conforme el Dispositivo es más pequeño, siempre suponiendo que estos tienen mayor resolución

12px (1em) para Dispositivos mayores a 1080 px. No olvidar definir el tamaño default en 12px

24px (2em) para Dispositivos mayores a entre 800px y 1080px

36px (3em) para Dispositivos menores a 800 px
